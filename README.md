
# 🛠️ Servis SalePage Builder

Servis SalePage Builder adalah sebuah sistem yang dibangunkan untuk membantu pengguna membina halaman jualan (sales page) dengan mudah, cepat dan fleksibel tanpa memerlukan kemahiran coding yang tinggi. Projek ini sesuai untuk digital marketer, freelancer, usahawan online dan developer yang ingin menghasilkan landing page dengan pantas.

## 🚀 Features
- Drag & drop / modular page builder  
- Customizable design (text, image, button, layout)  
- Responsive (mobile, tablet, desktop)  
- Fast rendering untuk landing page  
- Simpan & load konfigurasi page  
- Export HTML / deployment-ready page  

## 🏗️ Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Framework (optional): React / Vue / Vanilla JS  
- Backend (optional): Node.js / PHP / Firebase  
- Storage: JSON / Database  

## 📦 Installation
git clone https://github.com/Munzir-Mdn/servis-salepage-builder.git  
cd servis-salepage-builder  
npm install  

## ▶️ Usage
npm run dev  
Buka aplikasi di browser dan mula bina sales page menggunakan komponen yang disediakan.
